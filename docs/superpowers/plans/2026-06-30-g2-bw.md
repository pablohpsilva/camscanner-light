# G2 B&W Filter Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a B&W binarization filter to the scan pipeline — a new `BwEnhancer` strategy using Otsu's auto-threshold, and a mutually exclusive toggle button in the review screen's AppBar alongside the existing grayscale toggle.

**Architecture:** `BwEnhancer` is a new `const`-constructible `ImageEnhancer` strategy that runs in a `compute()` isolate: grayscale → Otsu histogram threshold → binary JPEG. `CaptureReviewScreen` replaces its `bool _grayscale` with a `_EnhancerMode` enum (none / grayscale / bw) and adds a `Key('bw-toggle')` AppBar button. The save pipeline (`CameraScreen` → `SaveController` → `DriftDocumentRepository`) is **unchanged** — G1 already wired `ImageEnhancer` through.

**Tech Stack:** Flutter/Dart 3.12.2+, `image` 4.9.1 (already a dep), `flutter/foundation.dart compute()`, `bdd_widget_test` + `build_runner` for BDD.

## Global Constraints

- Flutter/Dart 3.12.2+; `image` package 4.9.1 — **no new dependencies**.
- `img.bakeOrientation(image)` and `img.grayscale(src)` take **positional** first args.
- `img.grayscale(src)` **mutates in place** (does not return a new Image).
- `img.threshold(src, t)` **returns a new Image** — assign it: `final bw = img.threshold(oriented, t)`.
- JPEG encode quality **92** everywhere.
- All CPU-intensive work runs in **`compute()`** (never on UI thread).
- Enhancement failure is **silent** — the save pipeline catches all enhancer exceptions (already implemented in G1's `DriftDocumentRepository`).
- Toggle button keys: **`Key('grayscale-toggle')`** (existing), **`Key('bw-toggle')`** (new).
- B&W button uses `Icons.contrast` with `color: Theme.of(context).colorScheme.primary` when active (no `_outlined` variant assumed).
- Grayscale and B&W are **mutually exclusive** — the `_EnhancerMode` enum encodes this.
- No DB schema change in G2.
- TDD: write the failing test first, observe red, then implement, observe green.

```dart
import 'dart:typed_data';

import 'package:flutter/foundation.dart' show compute;
import 'package:image/image.dart' as img;

import 'image_enhancer.dart';

/// Binarizes a JPEG to pure black-and-white using Otsu's automatic threshold.
/// Runs in a [compute] isolate — never blocks the UI thread.
class BwEnhancer implements ImageEnhancer {
  const BwEnhancer();

  @override
  Future<Uint8List> enhance(Uint8List bytes) => compute(_bwFn, bytes);
}

// Top-level function required by compute() (must be isolate-sendable).
Uint8List _bwFn(Uint8List bytes) {
  try {
    final decoded = img.decodeImage(bytes);
    if (decoded == null) return bytes;
    // bakeOrientation: EXIF scrubber keeps the Orientation tag; encodeJpg
    // strips EXIF, so orientation must be baked into pixels first.
    // For already-baked flat bytes (post-warp), this is a safe no-op.
    final oriented = img.bakeOrientation(decoded);
    img.grayscale(oriented);                    // mutates in place
    final t = _otsuThreshold(oriented);         // automatic split — no magic numbers
    final bw = img.threshold(oriented, t);      // returns a new Image
    return Uint8List.fromList(img.encodeJpg(bw, quality: 92));
  } catch (_) {
    return bytes;
  }
}

// Otsu's method: finds the threshold that maximises inter-class variance
// between background and foreground. Operates on a grayscale Image where
// all channels are equal — reads pixel.r as the luminance value.
int _otsuThreshold(img.Image src) {
  final hist = List<int>.filled(256, 0);
  for (final px in src) {
    hist[px.r.toInt()]++;
  }
  final total = src.width * src.height;
  double sum = 0;
  for (var i = 0; i < 256; i++) sum += i * hist[i];

  double sumB = 0, maxVar = 0;
  int wB = 0, threshold = 128;
  for (var t = 0; t < 256; t++) {
    wB += hist[t];
    if (wB == 0) continue;
    final wF = total - wB;
    if (wF == 0) break;
    sumB += t * hist[t];
    final mB = sumB / wB;
    final mF = (sum - sumB) / wF;
    final v = wB.toDouble() * wF * (mB - mF) * (mB - mF);
    if (v > maxVar) {
      maxVar = v;
      threshold = t;
    }
  }
  return threshold;
}
```
